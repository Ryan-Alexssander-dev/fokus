<h1>Fokus</h1>
<p>&#9888; Projeto desenvolvido afim de estudos com Alura.<br></p>
<p>Para este projeto, foi fornecido todo o código HTML e CSS e arquivos de imagens e sons. Meu desafio foi estruturar todo o JavaScript da página para que ela funcionasse como no modelo apresentado previamente.</p>

<h2>Sobre o Projeto</h2>
<p>Este projeto é uma página web com a finalidade de ajudar com produtividade. Sua interface possui três cronômetros: foco, descanso curto e descanso longo. possui também sons reproduzidos ao iniciar o cronômetro, ao pausar e ao fim do tempo.</p>
